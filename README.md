# DFIR Phishing Campaign Case Study

## Overview
This repository contains a redacted DFIR-style analysis of a real-world phishing
campaign focused on credential harvesting via multi-stage redirection.

The analysis was conducted using passive techniques only and follows
responsible disclosure principles.

## Scope
- Phishing email analysis
- Redirect chain investigation
- Phishing backend behavior
- Base64 payload inspection
- VM isolation and OPSEC considerations

## Methodology
- Isolated virtual machine (VMware)
- Incognito / InPrivate browsing
- Browser DevTools (Network)
- Passive traffic observation
- No data submission, no active exploitation

## Contents
- `report/` – Full redacted analysis (PDF + Markdown)
- `screenshots/` – Evidence used in the report
- `notes/` – Methodology and analyst notes

## Disclaimer
All indicators have been partially redacted.
This repository is intended for educational and defensive purposes only.
